# FIN 233 — Class Portfolio & Macro Dashboard

Live dashboard for FIN 233 Equity Investments (Wake Forest University):
class portfolio performance, factor models, market & macro monitor, pitch
impact previews, and the pitch scoreboard.

Static, self-contained page — data is baked in and refreshed daily by an
automated job. Educational use; not investment advice. Data: Yahoo Finance,
FRED (St. Louis Fed), Ken French Data Library, Polymarket.
